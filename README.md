# Smart Tourism Platform

A full-stack tourism management web application designed to help users explore destinations, manage travel-related information, and improve the tourism experience through a modern web interface.

---

# Features

* User-friendly tourism platform interface
* Destination and travel information management
* Responsive frontend design
* Backend API integration using Spring Boot
* Database connectivity for storing tourism data
* REST API support
* Modern UI for better user experience

---

# Installation and Setup

## Clone Repository

```bash
git clone https://github.com/hiteshmhatre25/smart-tourism-platform.git
```

---

# Backend Setup

Move to backend folder:

```bash
cd tourism
```

Run Spring Boot application:

```bash
./mvnw spring-boot:run
```

For Windows:

```bash
mvnw.cmd spring-boot:run
```

---

# Frontend Setup

Move to frontend folder:

```bash
cd "frontend smart tourism"
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
npm start
```
